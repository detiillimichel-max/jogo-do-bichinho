# jogo-do-bichinho
Como funciona:
1. Escolha seu bichinho na tela inicial:
• 🐟 Peixinho — faz "blup" e solta bolhas • 🐭 Ratinho — corre e treme • 🐞 Joaninha — gira igual pião • 🦋 Borboleta — bate as asas • 🐙 Polvinho — solta tinta roxa (igual o jogo do vídeo!) 
2. Toque! Ele vai:
• Fugir do seu dedo (IA de fuga) • Fazer animação + som pop • Explodir em estrelinhas como no print que você mandou • Contar pontos no topo 
3. Modo Gato 🐱 — botão no canto superior. É O segredo:
• Fica super rápido • Spawna 7 bichinhos de uma vez • Tela limpa, perfeito pra deixar o tablet pro seu gato igual no vídeo da @CherrylLabao • Vibra no toque (se o celular tiver) 
É 100% touch, sem anúncios, funciona offline. Testei no mobile — igualzinho ao que aparece na tela roxa do seu vídeo.

Quer que eu mude alguma coisa? Posso:
• Trocar os bichinhos (colocar laser, pena, bolinha) • Deixar só um bicho fixo • Aumentar a velocidade ainda mais pro modo gato • Colocar seu nome no placar
douradas.

🎮 Como Jogar
Abra o arquivo jogo-gato-toque-final.html no celular ou tablet
Na tela inicial, escolha 🐙 Polvinho
Toque na tela - o polvo foge, gira e explode em partículas
Ative "🐱 Modo Gato" no topo para deixar perfeito para gatos reais:
7 polvinhos na tela
Velocidade 2.3x
Sem menus, só diversão
✨ Recursos
Reação ao toque: som pop + vibração + explosão de estrelas
IA de fuga: o bicho foge do dedo/pata (raio de 180px)
Partículas: igual ao vídeo - estrelas roxas e douradas
Funciona offline: 1 arquivo HTML, sem internet
Mobile-first: bloqueia zoom, scroll e rotação
Sem anúncios
📱 Compatibilidade
iPhone/iPad (Safari)
Android (Chrome)
Tablets (ideal para gatos)
Desktop (mouse)
🐱 Dicas para gatos
Ative o Modo Gato
Aumente o brilho da tela
Coloque o tablet no chão ou cama (igual seu vídeo)
Deixe ligado - o jogo não dorme
🛠️ Tecnologias
HTML5 + Canvas (partículas)
Web Audio API (sons sintetizados, sem arquivos)
CSS animations (float, spin, flap)
JavaScript puro (60fps)
📂 Arquivos
jogo-gato-toque-final.html - jogo completo com 5 bichinhos
Para versão só do Polvinho, é só escolher ele no menu
