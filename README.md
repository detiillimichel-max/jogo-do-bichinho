# jogo-do-bichinho
Como funciona:
1. Escolha seu bichinho na tela inicial:
• 🐟 Peixinho — faz "blup" e solta bolhas • 🐭 Ratinho — corre e treme • 🐞 Joaninha — gira igual pião • 🦋 Borboleta — bate as asas • 🐙 Polvinho — solta tinta roxa (igual o jogo do vídeo!) 
2. Toque! Ele vai:
• Fugir do seu dedo (IA de fuga) • Fazer animação + som pop • Explodir em estrelinhas como no print que você mandou • Contar pontos no topo 
3. Modo Gato 🐱 — botão no canto superior. É O segredo:
• Fica super rápido • Spawna 7 bichinhos de uma vez • Tela limpa, perfeito pra deixar o tablet pro seu gato igual no vídeo da @CherrylLabao • Vibra no toque (se o celular tiver) 
É 100% touch, sem anúncios, funciona offline. Testei no mobile — igualzinho ao que aparece na tela roxa do seu vídeo.

Quer que eu mude alguma coisa? Posso:
• Trocar os bichinhos (colocar laser, pena, bolinha) • Deixar só um bicho fixo • Aumentar a velocidade ainda mais pro modo gato • Colocar seu nome no placar
